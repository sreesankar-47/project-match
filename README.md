# ProjectMatch 🚀
### An AI-Powered Collaborative Project & Team Marketplace

ProjectMatch is a self-driven campus platform built on the **MERN Stack** (MongoDB, Express, React, Node.js) integrated with **Google Gemini AI**. Instead of automatically forcing students into teams, the platform acts as an open marketplace where students can discover projects that demand their unique skill sets, request to join teams organically, and allow project leads to review and accept members.

---

## 🛠️ Tech Stack & Architecture

* **Frontend:** React.js, Tailwind CSS (User Dashboard, Project Boards, Request Management)
* **Backend:** Node.js, Express.js (REST APIs, Request Handling, AI Routings)
* **Database:** MongoDB Atlas via Mongoose (Persistent storage for Users, Projects, and Applications)
* **AI Engine:** Google Gemini 2.5 Flash SDK (Smart skill demand matching, recommendation summaries)
* **Security:** Dotenv (Environment variables management for secret API keys)

---

## 💡 Core Workflow & Features

1. **Project Marketplace:** Project leads can post their project ideas, descriptions, and explicitly specify the technical skills demanded for the team.
2. **Skill-Demand Discovery:** Students browse active projects. The platform analyzes their profile skills against project demands, highlighting perfect fits.
3. **The Handshake System:** Students organically request to join projects that interest them.
4. **Lead Evaluation Panel:** Project leads receive join requests, view incoming student profiles, and choose to accept or decline the request to form their team.
5. **AI Discovery Assistant (Gemini):** * **For Applicants:** Explains *why* a project matches their skillset and how they can contribute.
   * **For Leads:** Summarizes an applicant's skill strengths relative to the project requirements to help the lead make fast, informed decisions.
